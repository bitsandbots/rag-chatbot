# Module Reference

## Starter Stack

### `rag_chatbot.rag_engine` â€” RAGEngine

Local RAG pipeline using ChromaDB for vector storage and Ollama for embeddings and generation.

#### Class: `RAGEngine`

```python
class RAGEngine:
    def __init__(
        self,
        collection_name: str = "documents",
        chroma_path: str = "./chroma_db",
        embed_model: str = "nomic-embed-text",
        gen_model: str = "qwen3:1.7b",
    ) -> None: ...
```

Creates a `chromadb.PersistentClient` at `chroma_path` and gets or creates a collection named `collection_name` with cosine similarity (`hnsw:space: cosine`).

**Parameters**

| Name | Default | Description |
|---|---|---|
| `collection_name` | `"documents"` | ChromaDB collection name |
| `chroma_path` | `"./chroma_db"` | Directory for persisted ChromaDB data |
| `embed_model` | `"nomic-embed-text"` | Ollama model for generating embeddings |
| `gen_model` | `"qwen3:1.7b"` | Ollama model for text generation |

**Methods**

```python
def embed_text(self, text: str) -> list[float]: ...
```

Calls `ollama.embed(model=self.embed_model, input=text)` and returns the first embedding vector (768 floats for `nomic-embed-text`).

```python
def add_documents(self, texts: list[str], ids: list[str] | None = None) -> None: ...
```

Embeds each text and adds them to the ChromaDB collection. If `ids` is `None`, assigns sequential IDs `doc_0`, `doc_1`, etc.

```python
def query(self, question: str, n_results: int = 3) -> dict: ...
```

Embeds `question` and queries ChromaDB for the top `n_results` matching chunks. Returns the raw ChromaDB result dict containing `documents`, `ids`, `distances`, etc.

```python
def generate_answer(self, question: str) -> str: ...
```

Full pipeline: calls `self.query(question)`, assembles a prompt with the top-3 context chunks, and calls `ollama.generate()`. Returns the generated answer string.

**Usage**

```python
from rag_chatbot.rag_engine import RAGEngine

engine = RAGEngine(chroma_path="./my_db", gen_model="llama3")
engine.add_documents(["Ollama runs models locally."], ids=["doc-0"])
answer = engine.generate_answer("Does Ollama use the cloud?")
print(answer)
```

### `rag_chatbot.document_loader` â€” load_text_files

Reads `.txt` files from a directory and splits them into fixed-size character chunks for ingestion.

#### Function: `load_text_files`

```python
def load_text_files(
    directory: str, chunk_size: int = 500
) -> tuple[list[str], list[str]]: ...
```

Iterates over all `*.txt` files in `directory` (sorted). Each file is read in full and split into non-overlapping character chunks of at most `chunk_size` characters. IDs are formatted as `{filename}_{offset}`, e.g. `guide.txt_0`, `guide.txt_500`.

**Parameters**

| Name | Default | Description |
|---|---|---|
| `directory` | â€” | Path to directory containing `.txt` files |
| `chunk_size` | `500` | Maximum characters per chunk |

**Returns**: `(chunks, ids)` â€” two parallel lists ready to pass to `RAGEngine.add_documents()`.

**Usage**

```python
from rag_chatbot.document_loader import load_text_files
from rag_chatbot.rag_engine import RAGEngine

chunks, ids = load_text_files("documents/", chunk_size=500)
engine = RAGEngine()
engine.add_documents(chunks, ids)
```

### `rag_chatbot.api_server` â€” create_app

Flask application factory. Wires HTTP routes to a `RAGEngine` instance.

#### Function: `create_app`

```python
def create_app(rag: RAGEngine | None = None) -> Flask: ...
```

Creates and returns a `Flask` application. If `rag` is `None`, constructs a `RAGEngine` from environment variables (`COLLECTION_NAME`, `CHROMA_PATH`, `EMBED_MODEL`, `MODEL`). If a `RAGEngine` is passed directly, it is used as-is â€” this is the pattern used in tests.

**Routes registered**

| Method | Path | Handler |
|---|---|---|
| `GET` | `/health` | Returns `{"status": "ok", "engine": "local"}` |
| `POST` | `/ingest` | Calls `rag.add_documents(texts, ids)` |
| `POST` | `/query` | Calls `rag.generate_answer(question)` |

**Usage (production)**

```python
# __main__.py pattern
from dotenv import load_dotenv
from rag_chatbot.api_server import create_app

load_dotenv()
app = create_app()
app.run(host="0.0.0.0", port=5000)
```

**Usage (testing)**

```python
from rag_chatbot.api_server import create_app
from rag_chatbot.rag_engine import RAGEngine

engine = RAGEngine(chroma_path=str(tmp_path / "chroma"))
app = create_app(rag=engine)
app.config["TESTING"] = True
client = app.test_client()
```

## After Upgrade

### `rag_chatbot.fastapi_app` â€” FastAPI app

Production async backend generated by `scripts/upgrade.sh`.

The module-level `app` is a `FastAPI` instance. It constructs its own `RAGEngine` from environment variables.

**Routes**

| Method | Path | Description |
|---|---|---|
| `GET` | `/api/health` | Health check with `mode: production` |
| `POST` | `/api/ingest` | Index documents into ChromaDB |
| `POST` | `/api/query` | Non-streaming RAG query |
| `POST` | `/api/stream` | SSE streaming token-by-token response |

Start with uvicorn:

```bash
uvicorn rag_chatbot.fastapi_app:app --host 0.0.0.0 --port 7860
```

### `rag_chatbot.pdf_loader` â€” load_pdf

PDF ingestion generated by `scripts/upgrade.sh`. Requires `pymupdf`.

#### Function: `load_pdf`

```python
def load_pdf(
    path: str, chunk_size: int = 500
) -> tuple[list[str], list[str]]: ...
```

Extracts text from all pages of the PDF at `path` using `pymupdf`, concatenates them, then splits into character chunks. IDs follow the same `{filename}_{offset}` convention as `load_text_files`.

**Usage**

```python
from rag_chatbot.pdf_loader import load_pdf
from rag_chatbot.rag_engine import RAGEngine

chunks, ids = load_pdf("documents/manual.pdf")
engine = RAGEngine()
engine.add_documents(chunks, ids)
```

### `rag_chatbot.chat_history` â€” ChatHistory

SQLite-backed chat session storage generated by `scripts/upgrade.sh`.

#### Class: `ChatHistory`

```python
class ChatHistory:
    def __init__(self, db_path: str = "./chat.db") -> None: ...
```

Creates two tables on init if they do not exist: `sessions` (id, created_at) and `messages` (id, session_id, role, content, created_at). Role is constrained to `user` or `assistant`.

**Methods**

```python
def create_session(self) -> str: ...
```

Inserts a new session row and returns the UUID session ID.

```python
def add_message(self, session_id: str, role: str, content: str) -> None: ...
```

Appends a message to an existing session. Timestamps are UTC ISO 8601.

```python
def get_messages(self, session_id: str) -> list[dict]: ...
```

Returns all messages for a session as a list of dicts with keys `role`, `content`, `created_at`, ordered by insertion order.

**Usage**

```python
from rag_chatbot.chat_history import ChatHistory

history = ChatHistory(db_path="./chat.db")
session_id = history.create_session()
history.add_message(session_id, "user", "What is Ollama?")
history.add_message(session_id, "assistant", "Ollama runs LLMs locally.")
messages = history.get_messages(session_id)
# [{"role": "user", "content": "...", "created_at": "..."}, ...]
```
