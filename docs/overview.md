# RAG Chatbot — Project Overview

## Purpose

This repository is the supporting source code for the offline-first AI stack guide published at:

https://coreconduit.com/techlounge/guides/offline-first-ai-stack.html

It provides a clone-and-run starter stack for building a local Retrieval-Augmented Generation (RAG) chatbot. The system is designed to run entirely on a Raspberry Pi 5 or Pi 4 with no cloud dependencies — all inference runs locally via Ollama.

## Goals

- **Clone-and-run**: minimal setup steps from fresh checkout to working API
- **Offline-first**: no network calls to external AI providers; all models run locally via Ollama
- **Pi-native**: tested on Raspberry Pi 5 (8GB); works on Pi 4 (4GB minimum)
- **Upgradeable**: starter Flask stack extends cleanly to a production-grade dual-process architecture via `scripts/upgrade.sh`

## What Is RAG?

Retrieval-Augmented Generation combines a vector search index with a local language model. Documents are chunked and embedded into ChromaDB. At query time, the most relevant chunks are retrieved and passed as context to the LLM, grounding its response in your data rather than general training knowledge.
